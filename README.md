# 📊 Sales & Finance Analytics Report | Excel Project


## 📌 Project Overview

This project focuses on building Sales and Finance Analytics Reports in Microsoft Excel to analyze business performance across customers, markets, regions, and fiscal years.

The reports help businesses track sales trends, monitor financial performance, compare targets against actual sales, and support data-driven decision-making.

---

## 🎯 Objectives

- Customer Performance Report: Analyzes customer-wise sales performance and tracks yearly sales growth.

- Market Performance vs Target Report: Compares actual market sales against targets to measure performance and identify gaps.

- Profit & Loss (P&L) by Fiscal Year Report: Tracks yearly sales, costs, and profit performance.

- Profit & Loss (P&L) by Month Report: Monitors monthly financial trends and profitability analysis.

---

## 🛠 Technical Skills

-  Applied ETL (Extract, Transform, Load) processes for data cleaning and transformation.

-  Created dynamic Date Tables using Power Query.

-  Generated Fiscal Months and Quarters for time-based analysis.

-  Built and managed Data Model relationships.

-  Integrated supplementary datasets into the existing data model.

-  Used DAX formulas to create calculated columns and custom calculations.

---

## 📈 Results

-  Improved sales and financial performance analysis.

-  Compared actual sales against targets.

-  Identified customer and market growth opportunities.

-  Reduced manual reporting effort through dynamic reports.

-  Supported better financial planning and decision-making.
